1. System requirements
Codes are written using R language and Rstudio. Required R packages are cited in the context of the codes.

2. Installation guide
Rstudio installation guide refers to https://posit.co/download/rstudio-desktop/.
All required R packages can be installed using the "install.packages" function in R.
Typical install time on a "normal" desktop computer will be 30-60 minutes.

3. Demo and Instruction for use
3.1 Unzip the zip file named "Divergent_dryland_productivity_trends_260119-Dryland_and_Ocean.zip"
3.2 Rename the unzipped folder to a short-version name, i.e., project_test.
3.3 Set the working path to the folder, "project_test" for example, using setwd('/Path/to/the_folder')
3.4 Unzip the "R codes.tar.xz" file under the path "/Path/to/the_folder"
3.5 Open the code files ending with ".R" under the path "/Path/to/the_folder/R" in Rstudio.
3.6 Install all required R packages using install.packages function in Rstudio.
3.7 Download result data file "data_for_sharing.tar.xz" using the link inside the file "/Path/to/the_folder/Data/ReadMe.txt"
3.8 Unzip the data file "data_for_sharing.tar.xz" under the "/Path/to/the_folder/" and rename the folder to "Data"
3.9 R codes all good for testing.
3.10 Expected output: Results and Figures for Main Figures 1-4 in the manuscript.
3.11 Expected run time for all demo R files on a "normal" desktop computer will be 45-60 minutes.
